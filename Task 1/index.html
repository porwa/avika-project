My Projects

SIGN LANGUAGE TRANSLATOR

Sign Language Translator is an innovative application designed to convert sign language gestures into text in real time. Using computer vision and machine learning technologies, it accurately detects and interprets hand movements through a camera. The project aims to improve communication for deaf and hard-of-hearing individuals by making interactions more accessible and inclusive. It features a simple and user-friendly interface, ensuring ease of use for all users. This solution helps bridge communication gaps and promotes better understanding between people with different communication needs.

MORSE CODE CONVERTOR

A Morse Code Translator is a tool or application that converts normal text into Morse code and Morse code back readable text. Morse Code is a communication method that uses dots (.) , dashes (-) and spaces to represent letters ,numbers and symbols. “In this project, JAVA is used as the backend, and HTML, CSS, and JAVASCRIPT are used as the frontend.”Morse code is used in military communication, ships, aircraft, and emergency services to send messages using dots and dashes. It is also used in radio communication and signaling through light or sound.
